micro.rb awesome ruby libraries!

What is this?

micro.rb aims at gathering small well focused ruby libraries, that do one thing and do it well. We are concerned about focus and clarity. Other metrics such as the number of dependencies and the lines of code also matter

We are still figuring out exactly what a micro library is - if you want to chime in please check out this github issue.
